# Vision-Based Autonomous Line Following (PID Control)

This project demonstrates a real-time vision-based line following system inspired by RoboticsAcademy (JdeRobot).

## Features
- Webcam-based perception pipeline
- Region of Interest (ROI) filtering
- HSV-based line detection
- Noise reduction and smoothing
- PID control for stable tracking
- Line-loss detection and recovery

## Demo
https://youtu.be/nJGsnMGM68Y

## Tech Stack
- Python
- OpenCV
- NumPy

## Description
The system simulates a robot’s perception and control loop using a monocular camera. It processes visual input to detect a path and applies PID control to maintain alignment, with basic behavior handling for robustness.
